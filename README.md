# Tracker

- MERN Stack by Traversy

## Features

- Users:
  - Log in:
    `POST /api/users/login`
  - Register:
    `POST /api/users`
  - Get info of current logged in user(after logged in)
    `GET /api/users/me`
- Habits:
  - Fetch habits:
    `GET /api/habits`
  - Fetch habit by id:
    `GET /api/habits/:id`
  - Add habit:
    `POST /api/habits`
  - Edit habit:
    `PUT /api/habits/:id`
  - Delete habit:
    `DELETE /api/habits/:id`
- Tasks:
  - Fetch tasks:
    `GET /api/tasks`
  - Fetch habit by id:
    `GET /api/tasks/:id`
  - Add habit:
    `POST /api/tasks`
  - Edit habit:
    `PUT /api/tasks/:id`
  - Delete habit:
    `DELETE /api/tasks/:id`
